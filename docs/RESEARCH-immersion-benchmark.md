## 0. 한 줄 결론

naraka.cafe는 **가격 엔진·공정성·소수점 거래·뉴스 3등급·랭킹**이라는 "증권 앱의 코어"를 이미 토스급으로 갖췄다. 지금 부족한 건 기능이 아니라 **① 매일 다시 오게 만드는 습관 루프(미션·스트릭·알림)와 ② 손님끼리 엮이는 소셜 레이어, ③ 한 달을 하나의 이야기로 묶는 서사·이벤트 페이싱**이다. 이 셋이 몰입·재방문·"오늘 뭐 샀냐" 대화의 실제 엔진이다.

## 1. naraka.cafe가 이미 잘 하고 있는 것 (재확인)

토스·모의투자 대회의 몰입 장치 중 상당수를 이미 구현했거나 설계에 반영함:

| 몰입 장치 (벤치마크) | naraka.cafe 현황 |
|---|---|
| 소수점·금액 기준 거래 (진입장벽↓, 토스 핵심) | ✅ 매수 금액 기준·수량 6자리 소수점, 자본 100% 투입 (PRD §4.2) |
| 실시간 시세·차트 + 보간 연출 | ✅ 5분 틱 + 프론트 보간, 캔들+5분봉 (§4.2, §6) |
| 조작 불가 공정성 (상품 이벤트의 생명) | ✅ 모든 돈 계산 서버 트랜잭션, 사전 생성 경로 (아키텍처 §1) |
| 뉴스로 만드는 긴장·서사 | ✅ 공시/정식/찌라시 3등급, 장중 시간차 노출, 어드민 라이브 찌라시 (§5) |
| 변동성 이벤트 연출 (기대·놀람) | ✅ 점프 이벤트·VI·서킷브레이커·상하한 배지 (§4.3~4.4) |
| 수익률 랭킹·사회적 비교 | ✅ 총자산 순위 상위 20 + 내 등수 고정 (§6) |
| 재방문 동기 (일일 보상) | ✅ 매장 방문 보너스 1일 1회 (단, 오프라인 코드 의존 — 아래 3장) |
| 지정가 자동매매 (자리 비운 시간 전략) | ✅ 지정가 예약주문 (§4.5) — 토스 "주식모으기"의 나라카식 변형 |
| 초보자 온보딩·교육 | △ `/guide` 존재하나 정적 설명 위주 (아래 3장) |

**시사점:** 코어는 이미 상위 티어. 아래 제안은 "없는 걸 새로 만들기"보다 **있는 자산을 몰입 루프로 엮기**에 방점.

## 3. 몰입 강화 요소 — 카테고리별 제안 (근거 + 나라카 적용안)

### 3.1 습관 루프: 일일 미션 · 출석 스트릭 [P0]

**근거.** 토스 리텐션의 핵심은 "주식 출석 체크"다 — 매일 예측만 해도 소수점 주식을 주고, **연속 출석이 끊기면 1일차로 리셋**되는 손실 회피 구조로 매일 접속을 만든다. 러닝·만보기 연구(Rewley 2021)에서도 "이미 획득한 레벨을 잃을 위험"에 처한 참가자가 목표 달성률이 18.4%p 높았다 — 손실 회피는 **스스로 획득한 것**일 때 가장 강하게 작동.

**나라카의 문제.** 현재 유일한 재방문 동기인 방문 보너스는 **매장 오프라인 코드**에 100% 의존. 매장에 못 오는 날엔 앱을 열 이유가 없다 → 한 달 완주율 하락 위험.

**적용안.**
- **출석 스트릭 (온라인 전용, 매장 방문과 별개):** 하루 1회 앱 접속·매매 예측 시 소액 보너스(예: +5,000~20,000원, 밸런스 영향 최소 수준) + 연속일 카운트. 끊기면 리셋 or 부분 회복(아래 윤리 주의).
  - ⚠️ **상품 이벤트이므로 보너스 금액은 밸런스 시뮬에 반드시 반영.** 방문 보너스(+100만) 대비 훨씬 작게. 스트릭 보상은 "현금"보다 **배지·타이틀·꾸미기**로 주는 게 공정성상 더 안전.
- **일일 미션 (거래 횟수가 아닌 "학습·건전 행동"에 보상):** 모의투자 대회 연구의 핵심 교훈 — 거래 횟수만 보상하면 무의미한 스팸 매매를 유발(MarketWatch 사례). 대신:
  - "오늘의 뉴스 3건 읽기" / "관심종목 1개 등록" / "포트폴리오 점검" / "차트 지표 켜보기" 같은 미션.
  - 완료 시 스트릭 +1. 매매 자체를 강제하지 않아 수수료 스팸·조작 유인 없음.
- **목표 그래디언트 연출:** "3일 더 오면 [옥자의 특별 배지]" 식으로 다음 보상까지 남은 칸을 시각화 → 목표 근접 시 노력 증가 효과.

### 3.2 소셜 레이어: 토론 피드 · 찌라시 라이브 결합 [P0]

**근거.** 토스 몰입의 큰 축은 **종목토론방(종토방)** — 사회적 증거·소속감·감정 공유. "나만 손실 본 게 아니구나"가 이탈을 막고, 새 글·댓글 확인이 **변동 보상**처럼 재방문을 만든다. naraka의 비즈니스 목표 1.1 자체가 "손님들 사이 대화거리 제공" → **이 대화를 앱 안에서 일어나게** 하는 게 정확히 이 기능.

**소규모 참가자 우려 → 오히려 기회.** PRD가 호가창을 보류한 이유(사람 적어 잔량 빈약)와 달리, 토론 피드는 **사람이 적어도 채팅방처럼 밀도가 느껴진다.** 어드민 찌라시(§5.1)와 결합하면 "요괴 기자단"이 던진 소문에 손님들이 실시간 반응 → 라이브 연출 극대화.

**적용안.**
- **종목별 + 전체 피드:** 종목 상세(`/stocks/[code]`)에 짧은 코멘트/댓글. 닉네임은 이미 랭킹 공개용이라 그대로 사용.
- **찌라시 연동:** 어드민이 찌라시 발행 → 피드 상단 고정 + "이 소문 믿는다/못 믿는다" 반응 버튼. 55% 적중률이라 반응이 갈리는 재미.
- **경량 운영:** 욕설 필터(이미 닉네임용 존재 §7) 재사용, 신고·삭제는 어드민(§8)에 편입. 익명·무관리 방지.
- **감정 표현 프리셋:** 자유 타이핑 부담↓ + moderation 부담↓ → "존버 간다 / 익절 / 물렸다 / 옥자님 살려줘" 같은 세계관 이모트(UI 이모지 금지 규칙 준수 위해 **텍스트/일러스트 스탬프**로).

### 3.3 알림·푸시: 습관 루프의 트리거 [P0]

**근거.** 토스는 관심종목 가격·뉴스·출석 결과를 푸시로 알려 **큐→행동→보상** 루프를 완성. 알림은 정보 전달을 넘어 "곧 중요한 일이 벌어진다"는 미래 지향적 긴장, 그리고 "놓쳐도 알려주니 괜찮다"는 심리적 안전망 역할까지 한다.

**나라카 적합성.** 5분 틱·VI·서킷브레이커·장중 뉴스라는 **알릴 거리가 이미 풍부**. 모바일 웹이므로 **Web Push(PWA)**로 구현 가능(네이티브 앱 불필요 — 비목표 1.3 위배 안 함).

**적용안 (사용자가 켜고 끄는 자율 설정 필수 — 피로·FOMO 균형):**
- 관심종목 ±X% 급변 / VI 발동 / 상·하한 도달
- 내 등수 변동("한 계단 내려갔습니다" = 손실 회피 + 경쟁 심리 자극)
- 새 공시·정식뉴스, 어드민 찌라시
- 지정가 예약 체결 완료
- 스트릭 끊김 임박("오늘 안 오면 N일 연속 끊겨요")
- ⚠️ 알림 시간대·종류를 유저가 조절하게 → "내가 설정한 도우미"로 인식되게. 야간(자정 폐장) 도배 주의.

### 3.4 성취 배지·타이틀: 수익률 밖의 인정 [P1]

**근거.** 리더보드만 있으면 상위권만 동기부여되고 **중하위권은 학습된 무기력**으로 이탈. PBL(Points·Badges·Leaderboards) 연구·CFA 게이미피케이션 보고서 공통 권고: **다양한 지표를 가시화**해 각자 강점으로 경쟁하게 하라.

**적용안 (세계관 캐논 활용).**
- **플레이 스타일 배지:** "존버의 화신"(장기 보유), "단타 요괴"(거래 횟수), "잡주 도박사"(테마주 비중), "배당 수집가"(우량주 금요일 배당), "뉴스 사냥꾼"(찌라시 적중).
- **오너별 팬덤 타이틀:** 옥자/미호/멜/바나 계열주 집중 보유 시 "옥자 가문의 집사" 등 → 세계관 몰입 + 종목 애착.
- **순간 포착 배지:** VI 직전 탈출, 상한가 매도, 첫 수익 실현 등.
- 배지는 **현금 가치 0**(공정성) — 순수 명예·프로필 표시. 랭킹 옆 배지 노출로 "저 배지 갖고 싶다" 동기.

### 3.5 한 달 서사 · 주차별 이벤트 페이싱 [P1]

**근거.** 라이브 서비스 게임의 리텐션 = **주 단위 이벤트 사이클** + 비트(beat) 페이싱(강렬↔휴식 교차, 후반 고조). 30일을 밋밋하게 두면 중반 이탈. naraka는 어드민 수동 도구(찌라시·서킷브레이커·경로 재생성 §4.3)가 이미 있어 **운영만으로 서사 연출 가능**(개발 최소).

**적용안 — 30일 몰입 곡선 예시.**
- **1주차 (8/1~8/7) 개장 러시:** 온보딩·첫 매수 유도, 잔잔한 변동으로 학습. 신규 가입 집중.
- **2주차 첫 대형 이벤트:** "요괴 실적 시즌" — 특정 섹터 편향 몰아주기 + 관련 정식뉴스 집중.
- **3주차 급변장(클라이맥스 전조):** 어드민 서킷브레이커/깜짝 폭락·폭등 1~2회 → 순위 대격변, 역전 시작. "잡주몰빵" 역전 카드가 살아나는 구간(밸런스상 후반 꼬리 확대와 일치 §10).
- **4주차 (8/24~8/30) 결승 주간:** "마지막 결전" 배너, 등수 변동 알림 강화, 종료 카운트다운. 마지막 개장일 8/30 종가로 확정 → 최고조.
- **한정 요소로 FOMO:** "이번 주에만 등장하는 [멜라이 릴리 임상 결과] 이벤트" 식 시즌 한정 서사 → "지금 참여 안 하면 놓친다".

### 3.6 온보딩: 첫 매수까지 손잡고 [P1]

**근거.** 토스 온보딩은 "이미 가진 계좌로 몇 번 탭 → 계좌 개설"로 행동 장벽(Fogg 모델의 '능력')을 극단적으로 낮춤. 학습과 실전을 분리하지 않고 **작은 금액으로 바로 해보게** 한다.

**나라카 적용안.**
- 가입 직후 **인터랙티브 튜토리얼**: "옥카콜라 1주 사보기 → 시세 보기 → 팔아보기" 3스텝(가상 안내, 실제 잔고 사용 or 튜토리얼 모드).
- `/guide`의 정적 규칙을 **상황별 툴팁**으로 분산(상하한 배지에 "?", VI 배너에 설명 링크) → 필요할 때 학습.
- **첫 행동 보상:** 첫 매수 완료 시 축하 연출 + 배지 → 유능감 즉시 형성.

### 3.7 캐릭터 감정 디자인 [P2]

**근거.** 토스 고양이 키우기처럼 의인화 캐릭터는 감정 애착·서비스 호감을 만든다. naraka는 이미 옥자·미호·멜·바나 + 펫 5마리(시온·코코·규종·선아·수아)라는 **강력한 캐논 자산 보유**.

**적용안 (개발 가벼운 것부터).**
- 빈 상태·VI·서킷브레이커·수익/손실 화면에 캐릭터 **반응 일러스트**(이미 PRD §6.1이 지향). 큰 손실 시 "옥자가 안쓰럽게 본다" 등.
- 오너별 종목 상세에 해당 캐릭터 아바타 → 종목=인물 연결로 애착.
- (여유 시) 펫 성장 미니요소를 스트릭 보상과 연결 — 단 밸런스 무관한 순수 꾸미기.

## 4. ⚠️ 이 이벤트에 특화된 주의사항 (상품 걸린 공정 경쟁)

토스·상용 앱의 게이미피케이션을 그대로 복붙하면 안 되는 이유 — **naraka는 상품이 걸린 실제 경쟁**이다.

1. **모든 보너스/보상은 밸런스 시뮬(`npm run simulate`)에 반영.** 스트릭·미션 현금 보상이 순위를 왜곡하면 공정성 훼손. → **비현금 보상(배지·타이틀·꾸미기) 우선**, 현금 보너스는 소액·전원 동일 획득 가능하게.
2. **FOMO·손실 회피 넛지의 윤리 경계.** CFA·Bomnüter 연구 경고: FOMO+게이미피케이션은 과잉 거래·도박적 행동을 부추긴다. naraka는 매도 수수료 0.5%가 이미 스팸 억제 장치. 넛지는 "고위험 베팅 놓치지 마" 대신 **"뉴스 읽기·점검 놓치지 마"** 방향으로.
3. **소셜 트레이딩("따라 담기")은 소규모라 신중.** 토스도 국내 예치율 낮아 효과 제한적이었음. 참가자 적으면 상위권 1명 따라하기 쏠림 → 시세 왜곡·재미 반감. **P2로 미루고, 넣더라도 위험지표(변동성·최대손실) 병기.**
4. **스트릭 리셋의 가혹함 조절.** 완전 리셋은 하루 못 온 손님에게 큰 좌절 → 이탈. **부분 회복(미션으로 복구)** 권장. 매장 방문이 어려운 날의 배려.
5. **UI 이모지 금지 규칙 준수** (확정 선호). 배지·반응·스탬프는 **일러스트/텍스트**로. 위 문서의 이모지는 설명용일 뿐 UI 반영 아님.
6. **캐논 어휘 금지 준수.** 배지·미션·서사 문안에 저승·천계·도깨비·염라 등 확장 어휘 파생어 전면 금지(§3.2).
